# Take Home Assignment

## Objective
Build a basic GraphQL API server using NodeJS, TypeScript, Yoga GQL, Prisma, and Pothos
GraphQL to manage a simple list of tasks. This assignment aims to evaluate your understanding of
GraphQL principles, schema design with Pothos, TypeScript usage, and general backend
development practices.

## Reasoning
There are some new technologies in this stack that I need to get comfortable with before jumping into the code.
Although that's the case, I can always leverage past experiences and languages to streamline my build time.

### Tech Stack + Initial Knowledge on the Tool
1. NodeJS - Comfortable 
2. TypeScript - Comfortable
3. GraphQL -  limited knowledge, but similar to REST API
4. Yoga - New to me, but similar to express
5. Pothos  -  New to me, but similar to Schemas I built at startups using ZOD
6. Prisma -  limited knowledge, but I've used ORMs in the past like Mongoose
